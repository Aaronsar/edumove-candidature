🤖 Algorithme d'orientation — Critères internes

▼

⚙️ Comment fonctionne l'orientation automatique

Quand un étudiant soumet sa candidature, l'algorithme calcule automatiquement la meilleure destination (Italie ou Espagne) et l'université recommandée, en fonction de 3 critères : filière visée, niveaux de langue et moyenne des spécialités de Première.

LINK Campus accepte tous les profils sans prérequis linguistique — cours d'italien intensif inclus la 1ère année. C'est l'option par défaut quand le profil ne correspond pas aux critères espagnols.

🎯 Cas particuliers Médecine/Dentaire

  • Espagnol B2 + notes > 15 → UCJC + UE Madrid/Malaga (profil fort)
  • Espagnol B2 + notes > 12.5 → UCJC + UE Valence/Alicante/Canaries
  • Espagnol B2 + notes < 12.5 → UCJC seule (entretien déterminant)
  • Anglais B1+ + notes > 15 → UE Madrid/Malaga (anglophone)
  • Anglais B1+ + notes < 12.5 → Redirection LINK
  • UCJC Médecine/Dentaire : 2 spécialités scientifiques requises

⚡ Pharmacie — Règles spéciales

  • LINK : acceptation automatique sans test — c'est la seule filière sans épreuve
  • UE & UCJC : critères académiques très bas (≥ 10)
  • Aucune spécialité scientifique requise (toutes les facs)
  • C'est l'argument de vente pour les profils non-scientifiques

⏱️ Transition automatique des statuts

Le statut passe automatiquement de "En cours" à "Réponse Edumove" après 2 heures ouvrables (9h-20h, 7j/7). Ce délai simule une validation humaine. La réponse personnalisée est générée automatiquement en fonction du profil de l'étudiant.

Un SMS est envoyé automatiquement à l'étudiant quand la réponse Edumove est prête (via SMS Factor). Le manual_override permet de court-circuiter le timer si besoin.

  • Mathématiques

  • Aucune spé requise : Toutes les filières LINK, UCJC Kiné, UCJC Pharmacie, UE Kiné, UE Pharmacie, UE Prépa Dentaire

📝 Format du QCM — Intégralement en français

Le test est composé de 80 questions réparties en 5 matières, toutes rédigées en français. Aucune connaissance de l'italien n'est nécessaire. Chaque bonne réponse vaut 1,5 point, chaque mauvaise réponse vaut -0,4 point. Les questions non répondues valent 0 point.

Score minimum pour réussir : variable selon la filière et le nombre de candidats. Pas de seuil fixe communiqué à l'avance.

📚 Programme détaillé des 5 matières

MatièreNb questionsThèmes abordésExemples de sujets
Biologie~18 questionsChimie du vivant, cellule, génétique, anatomie, physiologieStructure de l'ADN, mitose/méiose, système nerveux, circulation sanguine, enzymes
Chimie~12 questionsAtome, liaisons chimiques, réactions, acides/bases, chimie organiqueTableau périodique, pH, oxydo-réduction, nomenclature organique, stoechiométrie
Mathématiques~20 questionsAlgèbre, fonctions, géométrie, probabilités, statistiquesÉquations 2nd degré, dérivées, intégrales simples, probabilités conditionnelles
Physique~18 questionsCinématique, dynamique, fluides, thermodynamique, électricitéLois de Newton, pression hydrostatique, loi d'Ohm, optique géométrique
Culture générale~12 questionsCompréhension de texte, raisonnement logique, actualitéAnalyse de graphiques, syllogismes, compréhension de passages scientifiques

💡 Conseil à donner aux étudiants : réviser le programme de Terminale en sciences. Les questions sont de niveau bac, pas de niveau universitaire. La culture générale est souvent le facteur différenciant.

🧪 Tests d'admission UE (PE) — Détails complets

▼

📍 Format des 4 épreuves

  1. Test de langue
    Adapté à la langue du cursus choisi (espagnol, anglais ou français). Compréhension écrite et orale, grammaire, vocabulaire.
  2. Test de talent / aptitudes
    Raisonnement logique, aptitudes cognitives, résolution de problèmes. Type tests psychotechniques.
  3. Test de motivation
    Entretien individuel + questionnaire de personnalité. Évalue la motivation et le projet professionnel.
  4. Épreuve spécifique
    Contenu adapté à la filière choisie (biologie pour médecine, anatomie pour kiné, etc.).

⚡ Points clés pour l'équipe

  • Résultats sous 48h par email
  • En cas d'acceptation : 2 jours pour verser l'acompte de 2 500€
  • Les tests peuvent se passer en ligne ou en présentiel selon la session
  • Plusieurs sessions par an (voir deadlines ci-dessous)
  • Kiné FR Madrid : même si la filière est Hors PE, le cursus en français nécessite de passer les 4 épreuves

Le score au test est pondéré avec le dossier académique. Les moyennes indiquées sont des fourchettes basées sur les profils acceptés les années précédentes.

💡 Comment ça marche ?

L'année préparatoire (17 000€) permet de se préparer et d'intégrer directement le cursus Dentaire à Alicante l'année suivante (20 821€/an pendant 5 ans). Total sur 6 ans : 121 105€.

C'est l'option idéale pour les étudiants qui veulent faire dentaire en anglais mais qui n'ont pas le profil pour intégrer directement. Le B2 en anglais est la seule condition d'admission — pas de test PE, pas de moyenne minimum.

🤖 Réponse automatique — Comment ça marche

▼

⏱️ Déclenchement du timer

  1. L'étudiant confirme ses choix d'universités dans son espace
  2. Le champ uni_selections_at est enregistré en base
  3. Un timer démarre : 120 minutes ouvrées
  4. Horaires ouvrées : 9h–20h, 7j/7 (= 11h/jour)
  5. À l'expiration → statut passe automatiquement à reponse_edumove

✍️ Génération de la réponse

La réponse est générée par la fonction generateReponseEdumove() basée sur :

  • Score académique (interprétation : <10 = fragile, 10–13 = solide, 13+ = excellent)
  • Niveaux de langues (espagnol, anglais, 5 niveaux)
  • Filière (pharmacie, kiné, médecine, dentaire…)
  • Destination (Espagne / Italie)
  • Universités sélectionnées par l'étudiant

🔧 Déclenchement du timer — cas particulier

Si l'étudiant n'a pas encore fait ses choix d'universités (uni_selections_at = null), le système estime l'heure de réponse à partir de created_at + 2h. La réponse générée est stockée dans auto_response_text en base et affichée à l'étudiant.
